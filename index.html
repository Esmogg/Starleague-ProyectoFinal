¿Que es?

Starleague es un Sistema de Gestión Deportiva Modular (SGDM), una plataforma web orientada a la organización, administración y seguimiento de competencias deportivas, mentales y electrónicas. El sistema permite gestionar para cada tipo de competencia.

Se desarrollará utilizando PHP para el backend. HTML, CSS y JavaScript para el frontend y MySQL como gestión de base de datos relacional. La arquitectura seguirá el patrón MVC (Modelo Vista-Controlador).

¿Cuales son las funcionalidades?

El SGDM contempla tres tipos de competencias: Liga (Sistema todos contra todos con tabla acumulativa de puntos), Eliminación directa (el perdedor queda eliminado) y Sistema Suizo (emparejamiento por rendimiento acumulado). Cada formato tiene su propia lógica de generación de enfrentamiento y cálculo de posiciones, todo automatizado gracias al sistema.

Estará orientado a clubes deportivos, centros educativos, organizadores independientes y comunidades de deportes electrónicos o de mesa, permitiendo administrar participantes, rondas, resultados y tabla de posiciones desde el panel centralizado.
